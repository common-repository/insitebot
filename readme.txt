== Description ==

Get Insite and automate your customer conversion effortlessly. InsiteBot is a personalized, AI-powered chatbot for your website. It's as easy to set up as editing a spreadsheet.

== Installation ==

Simply install the InsiteBot plugin from your "Plugins > Add New" screen in WordPress. Otherwise, follow the steps below to manually install InsiteBot on your website.

1. Upload the InsiteBot plugin .zip file to the `/wp-content/plugins/` directory, and extract it.
1. Activate InsiteBot through the 'Plugins' screen in WordPress.
1. Use the InsiteBot icon on your WordPress dashboard screen to configure customize its appearance.

== Frequently Asked Questions ==

= How do I customize InsiteBot's appearance? =

Once you've followed the instructions (included in the "getting started" email) to generate your styling code, in your WordPress dashboard, navigate to "Appearance > Customize" and click the "Additional CSS" button. in your WordPress dashboard. Paste your generated code in the space provided and click "Publish". You should now see the changes reflected on your website.

= Is InsiteBot free? =

No, while it is free to install the WordPress plugin, InsiteBot will not work on your website unless you've signed up at [insitebot.com](https://insitebot.com/).
